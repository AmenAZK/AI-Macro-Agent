Goal: Create a modern landing page with a hero section, features list, and contact form.

Requirements:
1. Hero Section:
   - Large headline with gradient text
   - Subheading with clear value proposition
   - Call-to-action button with hover effect
   - Background with subtle gradient or pattern

2. Features Section:
   - 3-column grid layout
   - Icons for each feature
   - Brief descriptions
   - Consistent spacing and alignment

3. Contact Form:
   - Clean, minimal design
   - Input fields for name, email, message
   - Submit button with loading state
   - Form validation

Technical Requirements:
- Use HTML5 and CSS3
- Implement responsive design
- Add smooth animations
- Ensure accessibility
- Optimize for performance

Success Criteria:
- Visual match with design: >90%
- Responsive on all devices
- Smooth animations
- Working form validation
- Accessibility compliance
